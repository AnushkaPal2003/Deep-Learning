🐶🐱 Cat vs Dog Image Classification (CNN)
📌 Overview
Deep learning project using CNN to classify images of cats and dogs using TensorFlow/Keras.
🚀 Tech Stack
Python, TensorFlow, Keras, OpenCV, NumPy, Matplotlib
📂 Dataset
Kaggle Dogs vs Cats dataset (25,000 images, 2 classes)
🧠 Model


CNN with Conv2D (32→64→128→256)


BatchNormalization + MaxPooling


Dense layers + Dropout


Sigmoid output layer


🔧 Preprocessing


Image resize: 128×128


Normalization (0–1 scaling)


Data Augmentation (Flip, Rotation, Zoom)


📊 Results


Training Accuracy: ~92%


Validation Accuracy: ~85%


📌 Features


Binary image classification (Cat/Dog)


Data augmentation for better generalization


OpenCV-based image prediction
